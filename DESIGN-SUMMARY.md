# Design summary

- Design concept: Provider proof desk
- Layout personality: luxury corporate
- Typography: Bodoni 72 headings with Inter body text
- Color palette: primary #422006, secondary #fbbf24, accent #2563eb, background #fffbeb, text #1c1917, muted #78716c
- Shape language: Minimal use of containers
- Visual motif: premium dividers and proof plaques
- Unique layout choices: first sections use Provider proof desk, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
